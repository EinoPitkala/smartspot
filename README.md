## Ominaisuudet
- Helppokäyttöinen UI ([shadcn/ui](https://ui.shadcn.com))
- Monta aluetta
- Varttisähkö
- Nopea ja kevyt

## Pyöritä lokaalina
- `git clone https://github.com/EinoPitkala/smartspot`
- `cd smartspot`
- `npm i && npm run dev`
